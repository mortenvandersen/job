# Job Posts

Add job postings here for assessment and tracking.

## Naming Convention
`<company>-<role-title>.md` — e.g., `stripe-engineering-manager.md`

## Template
Each file should include:
- Source URL (if applicable)
- Date found
- The full job description (paste it in)
- Any notes or first impressions

## Status Tracking
Use a tag at the top of each file to track status:
`Status: reviewing | applying | applied | interviewing | closed | rejected | offer`
